俄罗斯方块[5]

發表於 2014-11-22   |   分類於 Game ， Cocos2dx

任务完成！

总的来说，俄罗斯方块还是比较好实现的，也确实收获了很多。
中间遇到的困难不多，算是比较顺利吧。
毕竟是新人任务。另外，也跟自己在做之前借鉴了很多前人的经验有关。
具体实现的细节，我觉得没必要细说了。因为几乎就是把前面那篇的方法进行了一个实现而已。
游戏主体除了各小方块外，其实也就只有5个游戏元素:

  1. Tetromino
  2. nextTetromino
  3. holdTetromino
  4. ghostTetromino
  5. playField

其中当前块（Tetromino）和下一块（nextTetromino）以及暂存块（holdTetromino）和当前块（Tetromino)不过是在不断轮流的角色轮换罢了。这种方式我想应该还是省去了一直创建新对象的内存消耗的。

另外就是，又单独创建了一个单例类SuperRotationSystem来进行旋转处理。
处理方法还是和前文一样的，之所以单独提出来，无非是因为它们进行旋转操作盒旋转处理都一样不说，代码还非常冗余，干脆精简下提出来做统一处理。

游戏在我的NOTE3上运行还是比较完美的，可玩性也比较高，虽然就我自己玩。但我个人觉得还是比较好玩。

可以提高的地方呢：

  1. 分辨率适配
  2. 游戏性
  3. 代码组织
  4. 实现方法
  5. 没有记录保存功能 暂时可以用：
    UserDefault::getInstance()->setIntegerForKey("best_score", bestScore);
  6. 。。。
  7. 。。。

慢慢发现吧。
